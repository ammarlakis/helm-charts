Helm Charts Repository

Reusable Helm charts for deploying applications on Kubernetes, published as a standard Helm repository.

Add Repository

Configure Helm 
helm repo add ammarlakis https://ammarlakis.github.io/helm-charts/
helm repo update
Search charts 
helm search repo ammarlakis
Install chart 
helm install <release-name> \
  ammarlakis/<chart-name> \
  --namespace <namespace> \
  --create-namespace
